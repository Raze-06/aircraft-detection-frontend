✈️ AI-Powered Aircraft Detection System

📌 Overview
This project is an AI-powered web application that detects aircraft such as helicopters, fighter jets, and airplanes from images using a custom-trained YOLOv8 model. The system features a modular frontend for seamless user interaction and a domain-specific dataset to improve detection performance.

--------------------------------------------------------------------------------------------------------------------------------------------------------------------------

🚀 Key Features

- 🧩 Modular Frontend Architecture
  Built a scalable frontend using reusable components to handle image upload, detection processing, visualization, and results display workflows.

- 🗂️ Custom Dataset Creation 
  Curated a dataset of 5,000+ images by manually collecting and organizing helicopter, fighter jet, and aircraft data.

- 🤖 Model Training (YOLOv8) 
  Trained a YOLOv8 object detection model on the self-built dataset, ensuring domain-specific accuracy without relying on pre-built datasets.

- 📊 Detection & Visualization
  Integrated model outputs into the frontend to display detected objects with bounding boxes and confidence scores.
